## 核心数据结构设计
基本思路：
- 每个组件尽量符合`vnode`规范
- 用数组来组织数据（有序）
- 尽量使用引用关系，不用冗余

```javascript
{
  'project one': {
    title: '',
    setting: {/*可能需要的配置项*/},
    props: {/*页面body的一些设置*/},
    components: [
      {
        id: 'xxx1',
        name: 'xxx1',
        tag: 'image',
        attrs: {
          fontSize: '14px'
        },
        children: []
      },
      {
        id: 'xxx2',
        name: 'xxx2',
        tag: 'text',
        attrs: {
          fontSize: '14px'
        },
        children: null
      },
    ]
  }
  //当前选中的组件id
  activeComponentId: 'xxx'
}
```
